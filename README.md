# Engineering Operations Management Website

A professional, minimal website developed for a school project on **Operations Management in the Engineering Industry**.

## Project Overview
This project synthesizes research on:
- Core engineering operations and management systems.
- Scaling strategies from prototyping to mass production.
- Future trends in engineering for 2025 (AI, ML, Agentic Systems).
- The impact of engineering on global safety and sustainability.

## Features
- **Modern WordPress-style aesthetic**: Clean layouts with premium technical imagery.
- **Interactive Q&A Section**: Tabbed interface for exploring research questions.
- **Responsive Design**: Fully functional on mobile and desktop.
- **Interactive Animations**: Smooth scrolling and reveal-on-scroll effects.

## Setup
To host this on GitHub Pages:
1. Create a new repository on GitHub named `engineering-ops-project`.
2. Run the following commands:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/engineering-ops-project.git
   git branch -M main
   git push -u origin main
   ```
3. Enable GitHub Pages in the repository settings (Settings > Pages > Deploy from branch).
